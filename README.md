# Firmes Sant Jordi 2026

WebApp per consultar les firmes de llibres de Sant Jordi 2026 a Barcelona.

Cerca per autor, llibre o editorial. Filtra per ubicació i franja horària. Visualitza en llista o sobre el mapa de Barcelona.

## Començar

```bash
npm install
npm run dev
```

Obre [http://localhost:5173](http://localhost:5173) al navegador.

## Stack

- React + TypeScript + Vite
- Tailwind CSS
- Leaflet + OpenStreetMap
