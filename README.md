# Firmes Sant Jordi 2026

Web app to browse book signing events for Sant Jordi 2026 in Barcelona.

Search by author, book or publisher. Filter by location and time slot. View as a list or on a map of Barcelona. Available in Catalan and Spanish.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Deploy to Vercel

1. Import the repo at [vercel.com/new](https://vercel.com/new)
2. Vercel auto-detects Vite — just click Deploy
3. Done! Every push to `main` triggers a new deploy

## Stack

- React + TypeScript + Vite
- Tailwind CSS
- Leaflet + OpenStreetMap
- i18n: Catalan / Spanish
